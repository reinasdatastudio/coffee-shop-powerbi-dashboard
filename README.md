An interactive **Power BI** dashboard for analysing coffee shop sales performance, product trends, customer purchasing behaviour, and store operations using **Power Query** and **DAX**.

## Overview

This project demonstrates the development of a three-page **business intelligence dashboard** using **Power BI**.

The dashboard transforms transactional sales data into interactive visualisations that support business decision-making by highlighting revenue trends, product performance, customer purchasing behaviour, and store-level operational insights.

The project incorporates data preparation using **Power Query**, analytical modelling with **DAX**, and dashboard design following business intelligence best practices.

### Full report
[Coffee Shop Sales Analytics Dashboard (Power BI)](https://reinasdatastudio.github.io/coffee-shop-powerbi-dashboard/coffee-shop-powerbi.html)

## Dataset

**Source**

Maven Analytics Coffee Shop Sales Dataset

The dataset contains approximately **149,000 coffee shop transactions** across multiple store locations, including sales, products, store information, and transaction timestamps.

## Dashboard Features

### Sales Overview

* Total sales and transaction volume
* Monthly sales trends
* Sales by store location
* Product category performance
* Interactive month filtering

### Product Performance

* Best-selling products
* Product category analysis
* Sales by location
* Hourly purchasing behaviour
* Interactive category filters

### Store Performance

* Store comparison
* Average transaction value
* Hourly sales heatmaps
* Store-level sales trends
* Interactive store filtering

# Power BIを用いたコーヒーショップ売上分析ダッシュボード

## 概要

本プロジェクトでは、**Power BI**を用いて、複数店舗を展開するコーヒーショップの売上データを分析するインタラクティブなBIダッシュボードを作成しました。

Power Queryによるデータ前処理、DAXを用いた分析モデルの構築、およびPower BIによるダッシュボード設計を通して、売上推移、商品パフォーマンス、顧客の購買行動、店舗ごとの業績を可視化しています。

### **レポート**
[Coffee Shop Sales Analytics Dashboard (Power BI)](https://reinasdatastudio.github.io/coffee-shop-powerbi-dashboard/coffee-shop-powerbi.html)

## 主な機能

### 売上概要

* 売上・取引件数のKPI
* 月別売上推移
* 店舗別売上
* 商品カテゴリ別売上
* 月ごとのインタラクティブフィルター

### 商品分析

* 売れ筋商品
* 商品カテゴリ分析
* 店舗別売上
* 時間帯別購買傾向
* 商品カテゴリフィルター

### 店舗分析

* 店舗別業績比較
* 平均取引額
* 時間帯別売上ヒートマップ
* 店舗別売上推移
* 店舗フィルター

## 使用技術

## ダッシュボード概要

Power BIを用いて、コーヒーショップの売上や商品パフォーマンス、顧客の購買傾向、店舗ごとの業績をインタラクティブに分析できるBIダッシュボードを構築しました。利用者はフィルターを用いて自由にデータを探索し、データに基づく意思決定を支援できます。
